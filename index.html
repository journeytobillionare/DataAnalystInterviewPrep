Excel & Google Sheets Interview Questions & Answers

1. Difference between COUNT and COUNTA?

COUNT counts only numeric values.
COUNTA counts all non-empty cells.

2. Difference between SUMIF and SUMIFS?

SUMIF works with one condition.
SUMIFS works with multiple conditions.

3. What is VLOOKUP?

VLOOKUP searches vertically in a table and returns matching values.

Example:
=VLOOKUP(101,A2:C10,2,FALSE)

4. Why is XLOOKUP better than VLOOKUP?

XLOOKUP can search left and right.
It is more flexible and does not need a column number.

5. Explain INDEX MATCH

INDEX MATCH is more dynamic and powerful than VLOOKUP.

Example:
=INDEX(B2:B10,MATCH(101,A2:A10,0))

6. What is IFERROR?

IFERROR handles formula errors and displays custom messages.

Example:
=IFERROR(A1/B1,\"Error\")

7. What is ARRAYFORMULA?

ARRAYFORMULA performs calculations on entire ranges automatically.

Example:
=ARRAYFORMULA(A1:A10*B1:B10)

8. What is QUERY Function?

QUERY retrieves filtered and structured data similar to SQL.

Example:
=QUERY(A1:C10,\"SELECT A,B WHERE C>100\")

9. Difference between Excel and Google Sheets?

Excel is faster for large datasets.
Google Sheets is better for collaboration and cloud access.

10. What are Pivot Tables?

Pivot Tables summarize and analyze large datasets quickly.

11. How would you clean messy datasets?

Use TRIM, CLEAN, SUBSTITUTE, PROPER and remove duplicates.
